CLI tools for software, APIs, and autonomous agents.

Empower your agent

Install the meta-skill once, then let the agent choose the right CLI from the registry.

OpenClaw openclaw skills install cli-anything-hub
Nanobot nanobot skills install cli-anything-hub

Work directly from your shell

Install the hub locally and manage both harness CLIs and public CLIs with one command surface.

PyPI pip install cli-anything-hub
cli-hub listBrowse both registries
cli-hub searchFind by keyword
cli-hub installInstall from the right source
cli-hub infoInspect commands and metadata
cli-hub updateRefresh installed tools
cli-hub uninstallRemove cleanly
